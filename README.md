 ## Installing
  To clone the repo:
 ```
 cd project_folder
 git clone https://github.com/freddychannarvaez/react-minesweeper.git
 ```
 To install dependencies inside project_folder:
 ```
 npm install
 ```
 To run the project
 ```
 npm run dev
 ```
